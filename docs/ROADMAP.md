# Roadmap

## Completed: RC3A-RC3D

- App product layer and safe package lifecycle.
- Responsive product experience and shared visual system.
- Goal-first Solution recommendation.
- Preview of required Apps, resources, permissions, and starter work.
- Persistent account-owned Workspace creation.
- Workspace reopening and extension without a competing store.
- Tasks, results, memory, activity, connections, and recommendations in the workspace model.
- Release validation across build, TypeScript, acceptance, regression, desktop, mobile, authenticated, and guest paths.

## Next Product Gate

- Expand the Solution catalog only where the required capabilities and connections are verifiable.
- Continue more validated execution results into the same Workspace model.
- Refine cross-session continuation and next-step recommendations.
- Consolidate hybrid inference and durable memory choices behind stable product boundaries.

## Public Promotion Gate

- Confirm parity between the validated release and the intended public site.
- Publish only screenshots and claims tied to the promoted release.
- Add reliability or performance metrics only when reproducible benchmark artifacts exist.

## Separate Research Direction

Embodied AI, GO2, and UAV work remain research directions and are not part of the RC3D completion claim.
