# Performance

No latency, throughput, uptime, user-volume, or production-scale metric is claimed by this showcase.

The current priority is functional integrity:

- Clean dependency installation.
- Deterministic lint, typecheck, regression tests, and production build.
- Authenticated browser continuity.
- Correct account isolation.
- Honest failure and blocker behavior.
- Persistence of results across refresh and reconnect.

Performance numbers should be added only with a reproducible benchmark that identifies the candidate commit, environment class, dataset, test procedure, and limitations.
